## 数据源

### 免费数据源 (优先)
1. **Akshare** - 财务数据、行情数据
2. **Baostock** - 历史行情、复权数据
3. **Tushare免费版** - 基础财务数据

### 数据获取策略
```python
# 估值/质量/成长因子 - 来自财务数据
- 从 akshare.stock_financial_report_em 获取三大表
- 计算同比增长率、比率指标

# 动量/波动/技术因子 - 来自行情数据
- 从 baostock 获取历史K线
- 使用 pandas-ta 库计算技术指标

# 情绪因子 - 特色数据
- 换手率/成交额: 行情数据
- 融资余额: akshare.stock_margin_detail_szse
- 北向资金: akshare.stock_hsgt_hist_em
```

## 实施计划

- 创建 `services/factor_calculation.py`

- 升级因子筛选面板

### Step 4: IC分析和测试 (1天)
